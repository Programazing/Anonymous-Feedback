Anonymous Feedback

This form accepts confidential feedback without logins or personal identifiers.
Please do not include your name, email, employer, or other identifying details.

The time when feedback is sent is not recorded.
The database has one table called feedback with three columns: id, body, and reviewed.

Feedback can only be seen on Sundays to keep it anonymous.

The code can be viewed on GitHub

There is no receipt, reply, or follow-up link. Keep your message generic if you want stronger anonymity.
